# Secuencia didáctica ACP, IA y matemática emocional

Repositorio de materiales y código del trabajo de grado para optar al título de Matemática:

**"Secuencia didáctica para la enseñanza del Análisis de Componentes Principales fundamentada en el modelo 7E de Eisenkraft y en la matemática emocional, con apoyo de inteligencia artificial"**

María José Giraldo Cortés. Fundación Universitaria Konrad Lorenz. Bogotá, Colombia, 2026.

## Resumen

Este repositorio contiene los anexos del trabajo de grado: el formato de consentimiento informado utilizado en la sesión, el código fuente del sistema de reconocimiento emocional desarrollado en Python con TensorFlow y EfficientNetB0, y el tutorial interactivo en R con la librería learnr que materializa la secuencia didáctica para la enseñanza del Análisis de Componentes Principales (ACP).

## Estructura del repositorio

- `anexos/A_consentimiento/` Formato de consentimiento informado entregado a los estudiantes participantes.
- `anexos/B_reconocimiento_emocional/` Notebook de Google Colab con el sistema de reconocimiento facial de emociones basado en EfficientNetB0 entrenado sobre FER2013.
- `anexos/C_laboratorio_learnr/` Tutorial interactivo en R Markdown que implementa las siete fases del modelo 7E para la enseñanza del ACP.
